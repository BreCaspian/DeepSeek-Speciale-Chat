> [!TIP]
> 使用小技巧  
>
> - 点击聊天气泡可以折叠你或 DeepSeek 的聊天输出，便于查看长对话。  
> - 对于较难任务，推理链可能很长，如果中途输出中断，可以输入 **「继续」** 让模型接着往下推理，如此往复直至任务结束。  
> - 可以搭配其他 LLM 一起对比测评，效率更高。
